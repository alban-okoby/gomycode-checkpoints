# Library Management System

Design the core components of a Library Management System in JavaScript with a focus on modularity and reusability. The system should use ES modules, clearly separated components, and demonstrate design patterns in a Node.js environment

## project Structure
```
33-lms/
├── index.js                  # Main entry point
├── modules/
│   ├── book.js               # Book class
│   ├── member.js             # Member class
│   ├── librarian.js           # Librarian class
│   ├── transaction.js        # Transaction class
│   ├── factory.js            # Factory pattern
│   ├── strategy.js           # Strategy pattern (fine calculation)
│   ├── observer.js           # Observer pattern (notifications)
│   └── utils.js              # Utility functions
└── package.json
```

## Run the project
```
npm i && npm run start
```
